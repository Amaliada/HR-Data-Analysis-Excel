# HR-Data-Analysis-Excel
Excel-based HR data analysis project including data cleaning, transformations, dashboards, and scenario analysis.

## Overview
This project demonstrates a complete data analysis workflow in Microsoft Excel, including data cleaning, transformation, analysis, and visualization.

## Key Features
- Data Cleaning and Preparation
- Data Transformations (Age Group, Salary Band, Tenure)
- Lookup Functions (XLOOKUP, VLOOKUP, INDEX/MATCH)
- Pivot Tables and Charts
- Interactive Dashboard with slicers and KPI indicators
- VBA Macro for automatic dashboard refresh
- Advanced filtering and sorting using modern Excel functions
- What-If Analysis using Scenario Manager and Goal Seek

## Dashboard
The dashboard provides interactive insights into employee distribution, salary trends, and performance metrics.

## Scenario Analysis
Multiple salary scenarios were created and analyzed, with results summarized in a dedicated sheet.

## Tools Used
- Microsoft Excel
- Pivot Tables & Charts
- VBA (Macros)
- Data Validation
- What-If Analysis tools
